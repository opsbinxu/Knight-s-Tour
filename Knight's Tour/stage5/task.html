Description

It's time for you to try this puzzle yourself. Now you need to allow your program to make multiple moves and mark squares you have already visited.

This is a hard puzzle, so hopefully using Warnsdorff's algorithm can help you finish the puzzle. Note that the algorithm doesn't guarantee you will finish the puzzle, and you are free to ignore the algorithm. Also, not all boards have a solution!

Objectives

In this stage, you should alter your program to do the following:

  1. Set up the board as before.
  2. Once the game starts, show the board's state and number of possible moves using Warnsdorff's algorithm.
  3. Prompt the player for the knight's move.
  4. Check if the move is valid. If not, ask player for another move.
  5. Move the knight and mark with 'X'. Mark visited squares with '*'.
  6. Repeat the above until there are no more possible moves.
  7. If the player visits every square, congratulate the player.
  8. If the player does not visit every square, print the number of moves they made.
