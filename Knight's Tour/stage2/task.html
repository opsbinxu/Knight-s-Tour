Description

Although this puzzle is based on chess, you can actually try this puzzle on any sized board. The dimensions of the board are not constrained to an 8x8 grid. You can try smaller boards like 5x5, rectangular boards like 4x8, even non-rectangular boards with missing squares! To make things simple, we will only focus on rectangular boards. Note, the board is guaranteed to have a solution if the smallest dimension is at least 5. Smaller boards are not guaranteed to have a solution.

Objectives

In this stage, you should alter your program to do the following:

  1. Ask the user for the board's dimensions. The order of the values should be x columns and y rows.
  2. After taking the starting position, check that it is a valid position.
  3. If it is invalid, you should show an error message Invalid position!, then prompt the user for another starting position until the user enters a valid position.
  4. Draw the board with the knight.

The greater-than symbol followed by space (> ) represents the user input. Notice that it's not the part of the input.
